# Game

用C语言编写的一些小游戏，在linux下gcc编译(加-lncurses)通过

## 用到的库

- ncurses : 提供字符终端处理库

只用到ncurses里几个简单的输入输出api，基本可以不看ncurses文档情况下看懂代码

## 游戏种类

- Plane_War(飞机小游戏)
- Snake(贪吃蛇)
- Flappy_Bird(像素鸟)
